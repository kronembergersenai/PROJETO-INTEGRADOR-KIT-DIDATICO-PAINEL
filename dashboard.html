KIT DIDÁTICO

Visão Geral

Este projeto propõe o desenvolvimento de um Painel Didático de Comando com Comunicação Sem Fio. Ele foi idealizado para atender turmas com muitos alunos, onde a quantidade de máquinas físicas seria insuficiente ou cara. A solução integra CLPs, inversores de frequência, botoeiras, sinaleiras e um roteador para gerenciar conexões de diversos computadores em um único conjunto de hardware.

Objetivos do Projeto

  • Montar um painel elétrico com CLP Siemens S7-1200, botoeiras, sinaleiras e contatora.
  • Permitir controle do inversor Altivar por entradas digitais.
  • Controlar o inversor pela saída analógica de 4–20 mA do CLP.
  • Integrar um roteador para permitir que vários computadores acessem o painel (um por vez).
  • Fornecer um recurso didático para aulas práticas, sem necessidade de múltiplos kits por aluno.

Resumo Técnico

O painel didático terá:

  • CLP Siemens S7-1200 com módulos digitais e analógicos.
  • Inversor Altivar controlado via sinal analógico 4–20 mA e entradas digitais.
  • Roteador para criar uma rede interna que permita a conexão controlada de vários PCs.
  • Botoeiras e sinaleiras para simulação prática de comandos e sinalização.
  • Contatora para acionamento de cargas.

O objetivo técnico é construir uma infraestrutura modular, escalável e fácil de usar em aulas, facilitando a integração de conceitos de automação industrial.

Documentação e Guia

Este projeto está alinhado ao Guia Prático de Projetos Integradores (SENAI), que apresenta a metodologia, etapas e critérios para avaliação de projetos técnicos.

O dashboard serve como material de apoio ao instrutor, permitindo apresentar de forma visual e clara os objetivos, componentes e arquitetura do painel didático.
